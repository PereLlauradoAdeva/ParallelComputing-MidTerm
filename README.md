# Parallel Programming - Mid Term Project

Morphological Image Processing with OpenMP

## Dataset

Water Bodies Dataset - Satellite Images:
https://www.kaggle.com/datasets/franciscoescobar/satellite-images-of-water-bodies/discussion?sort=undefined

## Compilation

```bash
g++ -std=c++11 -O3 -fopenmp -o main.exe src/main.cpp
```

## Execution

```bash
./main.exe
```
