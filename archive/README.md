# Dataset Archive

Aquest directori conté el dataset utilitzat per al projecte de programació paral·lela.

## Water Bodies Dataset

El projecte utilitza el **Water Bodies Dataset** per processar imatges en paral·lel.

### Descàrrega del Dataset Complet

Per obtenir el dataset complet:
1. Descarrega'l des de la font original (afegeix l'enllaç aquí)
2. Extreu els fitxers al directori `archive/Water Bodies Dataset/`

### Mostra d'Imatges

El directori `sample/` conté 5 imatges de prova per verificar que el programa funciona correctament sense necessitat de descarregar tot el dataset.

## Estructura

```
archive/
├── README.md                          # Aquest fitxer
├── sample/                            # Imatges de prova
│   ├── water_body_8421.jpg
│   ├── water_body_8422.jpg
│   ├── water_body_8423.jpg
│   ├── water_body_8424.jpg
│   └── water_body_8425.jpg
└── Water Bodies Dataset/              # Dataset complet (no inclòs al repositori)
    └── Images/
        └── test/                      # Imatges de test
```

## Nota

El dataset complet no està inclòs al repositori Git per evitar que sigui massa gran. Només s'inclouen algunes imatges de mostra per poder executar proves ràpides.
